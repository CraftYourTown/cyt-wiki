# Ranks within Towns

## **Players with no town: (Nomad)**

/towny map\
/towny prices\
/towny time\
/towny universe\
/town online\
/town here\
/town new\
/town join\
/plot perm\
/town list\
/nation list\
/resident\
/town spawn (public towns)\
/g (chat)

## **Default Town Member:**

All "players with no town" permissions plus:\
/resident (all features)\
/plot claim\
/plot unclaim\
/plot forsale\
/plot notforsale\
/plot toggle (all features)\
/plot set perm\
/plot set reset\
/town leave\
/town deposit\
/town spawn\
/tc (chat)\


## **VIP**

All "Default Town Member" permissions plus:\
Tax-exempt\


## **Builder:**

All "VIP" permissions plus:\
Ability to build around the town\


## **Assistant:**

All "Builder" permissions plus:\
/town claim\
/town invite\
/plot (all features)\
/town toggle public\
/town vault\


## **Co-Mayor:**

All "Assistant" permissions plus:\
/town (all features)\
/towny top\
All abilities in claimed areas (overriding resident switch, build etc settings)\


## **Mayor:**

All "Co-Mayor" permissions plus:\
/t set mayor\
/n leave\
/t set homeblock\
/t invite accept \<n>&#x20;
